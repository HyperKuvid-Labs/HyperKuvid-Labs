1. Gesture Recognition Hand Cricket Game
This project uses computer vision and machine learning to detect hand gestures via a webcam in real-time. The game mimics traditional hand cricket, where player and computer gestures are compared to determine runs or outs. OpenCV and a trained ML model are used for gesture classification. The system maintains a score and updates it based on the game's rules. It offers an interactive, touchless gaming experience ideal for casual play.

2. Inventory Simulation
This system simulates inventory tracking, focusing on expiry dates and optimal container placement. MongoDB is used as the NoSQL database for scalable and flexible data storage. Prisma ORM helps interact with the database in a type-safe, efficient way. The simulation allows adding, updating, and querying items with metadata like shelf life and storage zones. It helps visualize and test inventory logistics digitally before real-world deployment.

3. PDF Summarizer
The project provides a web-based interface for users to upload PDF documents. Once uploaded, the backend processes the file and extracts text using parsing libraries. It then uses an LLM (e.g., OpenAI API) to summarize the content intelligently. The summary is returned to the frontend, giving users a concise version of the PDF. It's useful for quickly understanding large documents like research papers or reports.

4. OAuth Login System
This is a secure authentication system built using OAuth2 and JWT (JSON Web Tokens). Users can log in using third-party providers like Google or GitHub, reducing the need for separate passwords. OAuth2 handles secure token-based access delegation, while JWT ensures session integrity and user verification. The backend validates tokens, stores user sessions, and restricts access to protected routes. This setup is robust, modern, and commonly used in production web applications.
